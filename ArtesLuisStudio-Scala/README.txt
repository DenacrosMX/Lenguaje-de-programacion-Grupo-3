
ARTES LUIS STUDIO – VERSION FINAL EN SCALA

Este proyecto adapta el sistema web Artes Luis Studio para ser servido
completamente por una aplicación desarrollada en Scala.

PASOS PARA EJECUTAR:

1. Descomprimir el ZIP
2. Abrir la carpeta en IntelliJ IDEA
3. Importar como proyecto sbt
4. Ejecutar Main.scala
5. Abrir el navegador en:
   http://localhost:8080

RESULTADO:
- Se abre el sistema web completo (Inicio, Nosotros, Portafolio, Admin, etc.)
- El servidor web está implementado en Scala (Akka HTTP)
